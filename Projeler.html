Elektronik Sayaç Okuma Yazılımı:
Elektronik sayaçların üzerinde bulunan endeks değerlerinin optik port ile veya RS-485 seri port haberleşme IEC 62056-21 protokolü ile sayaçlar üzerinden kısa-uzun readout bilgilerinin alınması ve verilerin analizinin yapılabildiği Windows form uygulaması.(Tüm marka elektronik sayaçlarını desteklemektedir.)

EPİAŞ (Enerji Piyasası İşletme A.Ş) DGPYS sistemi Web Servis Entegrasyonu:
Token bazlı authentication yapısına sahip Soap response-request protokolüne ait Web servis entegrasyonu yapılarak Tüm Abone bilgilerinin çekilmesi ve güncellenmesi ve birçok methodun kullanılarak Abonelere ait işlemlerin Servis aracılığıyla gerçekleşmesini sağlayan Windows uygulaması.

Sağlık Bakanlığı Personel Yönetim Sistemi:
Sağlık Bakanlığı bünyesinde bulunan tüm personelin izin yönetim, CV yönetim ve eğitimlerinin yöneltildiği web uygulaması. Tüm yazılım mimarisi tarafımdan hazırlanarak SPICE dahilinde Asp.net Core ve Angular2 tabanlı olarak geliştirilmiştir.
